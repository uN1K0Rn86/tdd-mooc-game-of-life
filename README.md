# Game of Life

An implementation of the game of life for TDD mooc University of Helsinki.

## Prerequisites

You'll need a recent [Node.js](https://nodejs.org/) version. Then download this project's dependencies with:

    npm install

## Developing

Run tests once

    npm run test

Run tests continuously

    npm run autotest

Code reformat

    npm run format

## Running the program

```
node src/run.mjs <filepath> <generations>
```
